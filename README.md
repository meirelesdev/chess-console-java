# Chess Console Java

> Jogo de xadrez rodando no console, desenvolvido em Java com programacao orientada a objetos. Implementa todas as pecas e regras basicas do xadrez.

## Stack

- Java

## Prerequisitos

- JDK 11+

## Como executar

```bash
# Clone o repositorio
git clone https://github.com/meirelesdev/chess-console-java.git
cd chess-console-java
```
